# Démonstration du SDK Web Adobe Experience Platform et du réseau Edge

Regardez le SDK web Adobe Experience Platform et le réseau Edge en action, avec un seul appel à Adobe pour envoyer des données à Experience Platform, à Analytics, à Audience Manager et à Target. Pour plus d’informations, consultez la [Documentation du SDK Web de Adobe Experience Platform](https://experienceleague.adobe.com/docs/experience-platform/edge/home.html?lang=fr).
