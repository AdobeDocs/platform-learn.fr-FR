# 3.3 Offer Decisioning

Dans ce module, vous obtiendrez une présentation pratique des fonctionnalités d’Offer Decisioning de Adobe Journey Optimizer.

Adobe Journey Optimizer permet de créer des offres personnalisées et de les diffuser de manière orchestrée vers toutes les destinations liées à Adobe Journey Optimizer.

Avec Offer Decisioning, vous pouvez déterminer la meilleure option à partir d’un ensemble de choix disponibles. Ces options peuvent être des offres, des recommandations de produit, des composants de contenu pour une expérience web, des scripts de conversation et des actions à entreprendre.

## Objectifs d’apprentissage

- comprendre la terminologie de base utilisée dans le cadre d’Offer Decisioning ;
- Découvrez comment configurer le service applicatif Offer Decisioning pour qu’il impacte les parcours clients réels.

## Conditions préalables

- Accès à Adobe Journey Optimizer
