# 1.8.2 Créer une nouvelle ressource, la réviser et l’approuver

## 1.8.2.1 Vérifier les images de référence dans Frame.io

Accédez à [https://next.frame.io/](https://next.frame.io/){target="_blank"}. Cliquez pour ouvrir le dossier de votre projet.

![WF](./images/wffio1.png)

Vous devriez maintenant voir toutes les images de référence fournies dans Workfront. Le concepteur a désormais accès automatiquement à tous les fichiers qui ont été chargés dans Workfront, dans un environnement sécurisé.

![WF](./images/wffio2.png)

Cliquez sur **+**, puis sélectionnez **Nouveau dossier**.

![WF](./images/wffio3.png)

Saisissez le nom : `Final Deliverables` et appuyez sur **enter**. Ce dossier sera utilisé pour charger le document final qui sera créé par le designerL

![WF](./images/wffio3.png)

## 1.8.2.2 Créer une ressource avec Adobe Firefly Services et Adobe Express

>[!NOTE]
>
>Si vous préférez ne pas créer la nouvelle ressource vous-même, vous pouvez télécharger la version terminée [ici](./images/timetravelnow.png).

Accédez à [https://firefly.adobe.com/](https://firefly.adobe.com/){target="_blank"}. Saisissez le `a neon rabbit running very fast through space` d’invite et cliquez sur **Générer**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset1.png)

Plusieurs images sont alors générées. Choisissez l’image qui vous plaît le plus, cliquez sur l’icône **Partager** sur l’image, puis sélectionnez **Ouvrir dans Adobe Express**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset2.png)

L’image que vous venez de générer est alors disponible pour modification dans Adobe Express. Vous devez maintenant ajouter le logo CitiSignal sur l&#39;image. Pour ce faire, accédez à **Marques**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset3.png)

Vous devriez alors voir un modèle de marque CitiSignal. qui a été créé dans GenStudio for Performance Marketing s’affichent dans Adobe Express. Cliquez pour sélectionner un modèle de marque dont le nom contient `CitiSignal`.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset4.png)

Accédez à **Logos** et cliquez sur le logo **blanc** Citisignal pour le déposer sur l’image.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset5.png)

Positionnez le logo CitiSignal en haut de votre image, pas trop loin du milieu.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset6.png)

Accédez à **Texte**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset6a.png)

Cliquez sur **Ajouter votre texte**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset6b.png)

Saisissez la `Timetravel now!` de texte, modifiez la couleur et la taille de la police, définissez le texte en **Gras** afin d’obtenir une image similaire à celle-ci.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset6c.png)

Cliquez ensuite sur **Partager**.

![GSPeM](./../../workflow-planning/module1.2/images/gsasset7.png)

Cliquez sur **... Tout afficher**.

![Frame.io](./images/frameioasset1.png)

Faites défiler vers le bas et sélectionnez **Télécharger**.

![Frame.io](./images/frameioasset2.png)

Cliquez sur **Télécharger**.

![Frame.io](./images/frameioasset3.png)

Vous disposerez alors de votre ressource sur votre ordinateur local.

![Frame.io](./images/frameioasset4.png)

Remplacez le nom du fichier par `timetravelnow.png`.

![Frame.io](./images/frameioasset5.png)

## 1.8.2.3 la ressource dans Frame.io

Revenez à [https://next.frame.io/](https://next.frame.io/){target="_blank"} et ouvrez le dossier de votre projet.

![WF](./images/wffio5.png)

Cliquez sur **Télécharger**.

![WF](./images/wffio6.png)

Sélectionnez le fichier **timetravelnow.png** et cliquez sur **Ouvrir**.

![WF](./images/wffio7.png)

Vous devriez alors voir ceci.

![WF](./images/wffio8.png)

Modifiez le statut en **Révision requise** puis double-cliquez sur l’image pour l’ouvrir.

![WF](./images/wffio9.png)

Identifiez l’un des réviseurs ou une des réviseuses de votre environnement et ajoutez un message du type : `ready for your feedback on this one`.

![WF](./images/wffio10.png)

Le réviseur ou la réviseuse peut alors apporter des commentaires pour apporter des modifications ou confirmer que l’apparence semble correcte.

![WF](./images/wffio11.png)

## 1.8.2.4 Consulter la ressource dans Workfront

Pendant que l’équipe de conception effectue une itération sur la ressource qu’elle crée, le chef de projet dans Workfront peut suivre ce qui se passe. Revenez à Workfront. Actualisez la page.

![WF](./images/wffio12.png)

Le dossier qui a été créé dans Frame.io apparaît désormais dans Workfront. Cliquez dessus pour l’ouvrir.

![WF](./images/wffio13.png)

Vous devriez alors voir ceci. Pointez sur le fichier **timetravelnow.png** et cliquez sur **Détails du document**.

![WF](./images/wffio14.png)

En tant que gestionnaire de projet, vous pouvez désormais voir la version actuelle de cette image afin de savoir ce qui se passe et que tout cela est en cours de traitement. Cliquez sur **Ouvrir dans Frame.io**.

![WF](./images/wffio15.png)

Une nouvelle fenêtre s’ouvre alors pour afficher la ressource dans Frame.io.

![WF](./images/wffio15a.png)

## 1.8.2.5 Approuver la ressource

Dans Workfront, accédez à **Validations**, puis cliquez sur **Ajouter**.

![WF](./images/wffio16.png)

Ajoutez-vous en tant qu’approbateur, puis cliquez sur **Soumettre la demande**.

![WF](./images/wffio17.png)

Vous devriez alors voir ceci. Cliquez sur **Ouvrir la révision** pour accéder à Frame.io.

![WF](./images/wffio18.png)

Dans Frame.io, vous pouvez voir tous les commentaires et vérifier la ressource. Cliquez pour ouvrir le champ **Votre décision**.

![WF](./images/wffio19.png)

Sélectionnez **Approuvé**.

![WF](./images/wffio20.png)

Revenez à Workfront et actualisez la page. Vous verrez désormais que le statut ici est également modifié. La ressource est approuvée et peut ensuite être utilisée pour la diffusion et l’activation.

![WF](./images/wffio21.png)
