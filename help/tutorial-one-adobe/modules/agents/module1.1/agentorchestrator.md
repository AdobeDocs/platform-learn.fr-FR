# 1.1 Agent Orchestrator

## Objectifs d’apprentissage

Découvrez comment utiliser les agents Adobe Experience Platform et Agent Orchestrator pour :

- Analyse des tendances d’achat
- Identification des audiences à forte propension
- Validation des performances du parcours
- Création d&#39;un nouveau parcours pour le déploiement CitiSignal Fiber Max

## Conditions préalables

- Accès à Adobe Journey Optimizer et Customer Journey Analytics
- Familiarité avec [les concepts d’Agent Orchestrator](https://experienceleague.adobe.com/fr/docs/experience-cloud-ai/experience-cloud-ai/agents/agent-orchestrator)
- Accès aux données de démonstration CitiSignal : Achats, Commandes, Audiences, Parcours
