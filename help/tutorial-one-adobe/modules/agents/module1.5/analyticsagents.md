# Analytics et agents

## Objectifs d’apprentissage

Dans ce module, vous analyserez les données qui résident dans Adobe Analytics et/ou Customer Journey Analytics à l’aide d’agents externes autres qu’Adobe.

## Conditions préalables

- Accès à Adobe Analytics (AA) et/ou Adobe Customer Journey Analytics (CJA)
- Accès à ChatGPT et/ou Claude.ai avec abonnement payant

## Exercices

[1.5.1 CJA et ChatGPT avec le serveur MCP](./ex1.md){target="_blank"}

Dans cet exercice, vous allez configurer un serveur MCP personnalisé dans ChatGPT pour connecter directement ChatGPT à Customer Journey Analytics et vous commencerez à analyser vos données hébergées dans Adobe Customer Journey Analytics (CJA) directement dans ChatGPT.

[1.5.2 CJA et Claude.ai avec serveur MCP](./ex2.md){target="_blank"}

Dans cet exercice, vous allez configurer un serveur MCP personnalisé dans Claude.ai pour connecter Claude.ai directement à Customer Journey Analytics et vous commencerez à analyser vos données hébergées dans Adobe Customer Journey Analytics (CJA) directement depuis Claude.ai.

[1.5.3 Adobe Analytics et Claude.ai avec serveur MCP](./ex3.md){target="_blank"}

Dans cet exercice, vous allez configurer un serveur MCP personnalisé dans Claude.ai pour connecter Claude.ai directement à Adobe Analytics et vous commencerez à analyser vos données hébergées dans Adobe Analytics (AA) directement depuis Claude.ai.
