# Informations sur le prépartage de Correspondance de segments

Lorsque vous décidez avec un partenaire stratégique de partager des données, il est important de savoir comment vos clients l’accueilleront, afin que vous puissiez connaître l’utilité de ce partage de données. Correspondance de segments vous permet de saisir le chevauchement avec les partenaires de données potentiels avant de partager des données. Pour plus dʼinformations, consultez la [documentation](https://experienceleague.adobe.com/docs/experience-platform/segmentation/ui/segment-match/overview.html?lang=fr).
